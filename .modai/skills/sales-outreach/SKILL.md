# Sales Outreach

Use this skill when the user needs direct revenue help: outreach, follow-up, objection handling, or offer framing.

- Write with concrete commercial intent. Default to short messages that can be sent today.
- Prefer one clear CTA per draft.
- When the user sells software, agencies, services, or consulting, push toward booked calls, demos, or paid trials.
- If the user shares lead context, segment by lead temperature before writing copy.
- After giving the main output, add one stronger variant and one lower-friction variant.
