# Content Repurposer

Use this skill when the user has one long asset and wants multiple short outputs from it.

- Convert a source video, podcast, call, article, or document into short-form content ideas.
- Return platform-native outputs: hook, angle, caption, CTA, and posting format.
- Prioritize assets that can ship in 24 hours.
- For creator workflows, recommend 3-5 derivatives instead of one large content plan.
- Keep language crisp and publishable without extra cleanup.
