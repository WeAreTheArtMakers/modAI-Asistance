Quick Start
Install, unlock, choose a plan, connect a model, and run your first workflow.
  1. 1
    Move modAI to Applications
    Keep the app in Applications so updates, permissions, and future activation behave predictably.
  2. 2
    Allow the first launch
    If macOS blocks the app, use Open Anyway or keep Sentinel nearby as an optional Gatekeeper utility.
  3. 3
    Choose payment and activation
    The launch page should handle trial, card checkout, crypto checkout, and license delivery in one flow.
  4. 4
    Connect a model provider
    Start with Ollama for local use, then add Gemini or Anthropic only when you need cloud reasoning.
  5. 5
    Run a workflow pack
    Give new users a Founder, Sales, or Research starting pack so the first value appears in minutes.
Card billing Keep recurring plans simple

Use a merchant-of-record flow for monthly and yearly plans so tax, invoicing, and failed payment recovery stay out of the app binary.

Stablecoin checkout Offer USDC and USDT

Crypto should be available for annual plans, founder passes, or credit bundles with immediate license delivery after payment.

Activation Keep the first run friction low

Gatekeeper support note
